# Nails.S. Studio CRM - Light Theme V3 (Полная версия)

## Что изменено:

### Цветовая схема:
| Было (V3 Dark) | Стало (Light) |
|----------------|---------------|
| #8B5CF6 (фиолетовый) | #3B82F6 (синий) |
| #EC4899 (розовый) | #0EA5E9 (голубой) |
| #0f0a1e (темный фон) | #FFFFFF (белый) |
| #1e1b4b (темно-синий) | #F8FAFC (светло-серый) |
| #312e81 (темный) | #E2E8F0 (серый) |
| purple-600 | blue-600 |
| violet-400 | sky-400 |
| indigo-600 | blue-600 |

### Файлы для замены:
1. app/src/App.css → Светлые стили, белый фон
2. app/src/App.tsx → Синие акценты вместо фиолетовых
3. app/src/index.css → CSS переменные (светлая тема)
4. app/src/types/index.ts → Цвета услуг
5. app/src/components/AuthDialog.tsx → Светлая тема
6. app/src/components/TimelineGrid.tsx → Светлая тема
7. app/src/components/AppointmentForm.tsx → Светлая тема
8. app/src/components/ClientAnalytics.tsx → Светлая тема
9. app/src/components/CashRegister.tsx → Светлая тема
10. app/src/components/Notifications.tsx → Светлая тема

### Установка:
1. Распакуйте архив
2. Скопируйте все файлы в папку проекта с заменой
3. Установите зависимости: npm install
4. Запустите: npm run dev
5. Очистите кэш браузера (Ctrl+Shift+R)
